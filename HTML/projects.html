One Handed Rollator

ES1050 Engineering Design Course Project | January 2023 - April 2023

Created a Rollator Attachement so that a rollator can be used by one hand. This project was a partnership with St.Joseph's Hospital. Elderly impaired patients need to be able to fully use a rollator using only one hand because they lack strength in both hands which prevents them to evenly brake and steer.

Function Generator Course Project

ECE 2240A Project | November 2023

The main goal of this project is to acquire practical experience in designing an electronic function generator, with a specific focus on learning printed circuit board (PCB) design. The project encompasses the application of lab instruments, including a Digital Multimeter, Breadboard, Digital Oscilloscope, and Voltage Power Supply, along with electronic components like Resistors, Capacitors, and Op-amps, to ensure the functionality of the designed device. Key skills to be developed include the use of Micro-Cap for circuit design and simulation, as well as proficiency in Eagle software for PCB design. The project concludes with hands-on learning experiences in soldering and troubleshooting the designed function generator by locating any short circuits and voltage drops, providing a comprehensive skill set in electronic device development
